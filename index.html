101 TETRIS

Abstract

โปรเจ็กนี้เป็นการพัฒนาซอฟต์แวร์เกม โดยมีต้นแบบมาจากเกม Tetris ซึ่งมีชื่อโปรเจ็กว่า 101 TETRIS เพื่อนำความรู้จากวิชา Physical Computing มาปรับใช้ในการทำซอฟต์แวร์เกมจากภาษาคอมพิวเตอร์ C โดยตัวเกมจะมีบล็อกเลโก้ตกลงมาต้องเรียงโดยเอาชีวิตรอดให้นานที่สุดเพื่อเก็บคะแนน โดยการควบคุมจะใช้เพียงแค่ปุ่ม A และ D เท่านั้น

Details

- จะมีตัว lego ลงมาจากด้านบนให้เราควบคุมเพื่อจัดสรรพื้นที่ให้ได้มากที่สุด

- การควบคุมจะใช้ A เพื่อไปซ้าย D เพื่อไปขวา (มี delay ในการทำงาน เพราะฉนั้นค่อยๆกดนะครับ)

- ถ้ามี lego ชนและหยุดขยับ จะได้รับคะแนนเพิ่ม 10 คะแนน

- หาก lego ที่บังคับอยู่ชนและอยู่นอกพื้นที่ เกมจะจบทันที

- จะมีการบอก score ผู้เล่นเมื่อจบเกม

ขั้นตอนการทำงานของระบบ

- โดยเมื่อเริ่มเปิดเกมระบบจะเช็คว่าเกมจบหรือยัง ถ้ายังไม่จบให้อัปเดทแผนที่ ถ้าจบแล้วจะแสดงผลคะแนน

- ถ้าเกมยังไม่จบจะทำการอัปเดทข้อมูลของแผนที่ ตำแหน่งของตัวเลโก้

- รับค่า Input จาก Keyboard เป็นปุ่ม A หรือ D

- แสดงผลตำแหน่งของเลโก้บนแผนที่

- ดีเลย์0.5วินาที และจึงเคลียร์หน้าจอ

- ทำการเช็คว่าเกมจบไหมถ้าเกมจบแล้วแสดงผลคะแนน ถ้ายังไม่จบจะทำซ้ำต่อไปเรื่อยๆ

การทำงานส่วนใหญ่นั้นจะเป็นแก้ไข map[HEIGHT][WIDTH] และ นำไปวนลูปแสดงผล
การสร้าง lego

- จะทำการจองพื้นที่ใน memory

- แล้วกำหนดค่าต่างๆ (รูปแบบ, ตำแหน่ง)

- แล้วให้ตัวpointer (playingLego) ชี้ไปยัง address นั้นๆ
